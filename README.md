# GadgetHub

A modern React-based eCommerce web application built with Vite, featuring product browsing, search, filtering, wishlist management, cart functionality, product comparison, and personalized recommendations.

## Live Demo

🔗 https://gadgethub-tech-gadget.netlify.app/

## Features

* Product Search & Filtering
* Shopping Cart & Wishlist
* Product Comparison
* Recently Viewed Products
* Personalized Recommendations
* Dark/Light Theme
* Responsive Design

## Tech Stack

* React
* React Router
* Context API
* Vite
* Local Storage

## Installation

```bash
npm install
npm run dev
```
